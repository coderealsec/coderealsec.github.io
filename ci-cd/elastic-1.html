Elasticsearch Kurulumu: Rocky Linux ve Docker

Rocky Linux Üzerinde Native Kurulum

Rocky Linux'ta Elasticsearch'i native olarak kurmak için aşağıdaki adımları izleyin:

1. Elasticsearch Deposu Ekleyin

Elasticsearch paketlerini kurabilmek için Elastic deposunu ekleyin:


sudo rpm --import https://artifacts.elastic.co/GPG-KEY-elasticsearch
cat <

NOT: Burada kullanılan "8.x", Elasticsearch'in ana sürüm serisidir. Daha yeni bir sürüm için Elastic'in resmi GitHub sayfasını kontrol edin.

2. Elasticsearch'i Kurun

Depoyu ekledikten sonra Elasticsearch'i aşağıdaki komutla kurabilirsiniz:


sudo dnf install elasticsearch

3. Elasticsearch Servisini Başlatın

Kurulum tamamlandıktan sonra Elasticsearch servisini başlatın ve sistem başlangıcında otomatik çalışmasını sağlayın:


sudo systemctl enable elasticsearch
sudo systemctl start elasticsearch

4. Elasticsearch Durumunu Kontrol Edin

Kurulumun başarılı olup olmadığını kontrol etmek için:


curl -X GET "localhost:9200/?pretty"

Docker ile Elasticsearch Kurulumu

1. Docker Compose Dosyası Oluşturun

Elasticsearch'i Docker üzerinde çalıştırmak için bir docker-compose.yml dosyası oluşturun:


version: '3.7'
services:
  elasticsearch:
    image: docker.elastic.co/elasticsearch/elasticsearch:8.x
    container_name: elasticsearch
    environment:
      - discovery.type=single-node
    volumes:
      - es_data:/usr/share/elasticsearch/data
    ports:
      - "9200:9200"
volumes:
  es_data:
    driver: local

NOT: Elasticsearch için kullanılan imaj sürümü "8.x" olarak belirtilmiştir. Daha yeni sürümleri kontrol etmek için Elasticsearch Docker GitHub reposunu ziyaret edin.

2. Elasticsearch'i Docker Compose ile Başlatın

Docker Compose kullanarak Elasticsearch'i başlatmak için şu komutu çalıştırın:


docker-compose up -d

Sonuç

Bu rehberde Rocky Linux için Elasticsearch'in hem native hem de Docker ortamlarında nasıl kurulacağını detaylı bir şekilde ele aldık. Daha yeni sürümleri düzenli olarak kontrol ederek sisteminizi güncel tutabilirsiniz.
