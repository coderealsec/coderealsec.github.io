Elasticsearch Kurulumu: Native ve Docker

Native Kurulum

Elasticsearch'i native olarak kurmak için aşağıdaki adımları izleyin. Bu yöntemle, Elasticsearch doğrudan işletim sisteminize kurulur.

1. Elasticsearch Deposu Ekleyin

Ubuntu için Elastic deposunu eklemek üzere aşağıdaki komutları çalıştırın:


curl -fsSL https://artifacts.elastic.co/GPG-KEY-elasticsearch | sudo gpg --dearmor -o /usr/share/keyrings/elastic.gpg
echo "deb [signed-by=/usr/share/keyrings/elastic.gpg] https://artifacts.elastic.co/packages/8.x/apt stable main" | sudo tee /etc/apt/sources.list.d/elastic-8.x.list
sudo apt update

2. Elasticsearch'i Kurun

Depoyu ekledikten sonra Elasticsearch'i kurabilirsiniz:


sudo apt install elasticsearch

3. Elasticsearch Servisini Başlatın

Elasticsearch servisini başlatmak için aşağıdaki komutları çalıştırın:


sudo systemctl enable elasticsearch
sudo systemctl start elasticsearch

Docker ile Elasticsearch Kurulumu

1. Docker Compose Dosyası Oluşturun

Elasticsearch'i Docker kullanarak çalıştırmak için bir docker-compose.yml dosyası oluşturun:


version: '3.7'
services:
  elasticsearch:
    image: docker.elastic.co/elasticsearch/elasticsearch:8.x
    container_name: elasticsearch
    environment:
      - discovery.type=single-node
    volumes:
      - es_data:/usr/share/elasticsearch/data
    ports:
      - "9200:9200"
volumes:
  es_data:
    driver: local

2. Elasticsearch'i Başlatın

Docker Compose ile Elasticsearch'i çalıştırmak için şu komutu kullanabilirsiniz:


docker-compose up -d

Sonuç

Elasticsearch'i hem native hem de Docker ile kurabilirsiniz. Bu adımlarla hızlıca bir Elasticsearch ortamı oluşturabilirsiniz. Herhangi bir sorun yaşarsanız destek alabilirsiniz.
