Elasticsearch Kurulum Rehberi

Elasticsearch Native Kurulumu (Rocky Linux)

1. Adım: Java Kurulumu

Elasticsearch, Java’ya ihtiyaç duyar. Gerekli JDK sürümünü yükleyin:

sudo dnf install java-11-openjdk

2. Adım: Elasticsearch Reposunu Ekleyin

Elasticsearch için resmi Elastic deposunu ekleyin:

sudo rpm --import https://artifacts.elastic.co/GPG-KEY-elasticsearch
sudo tee /etc/yum.repos.d/elasticsearch.repo <

3. Adım: Elasticsearch Yükleyin

Elasticsearch’i yükleyin ve çalıştırın:

sudo dnf install elasticsearch
sudo systemctl enable --now elasticsearch

4. Adım: Elasticsearch Yapılandırması

/etc/elasticsearch/elasticsearch.yml dosyasını düzenleyerek gerekli ayarları yapın:

network.host: 0.0.0.0
discovery.type: single-node

Elasticsearch Docker Kurulumu

1. Adım: Elasticsearch Docker Image’ını İndirin

Elasticsearch için en güncel Docker image’ını indirin:

docker pull elasticsearch:8.10.0

2. Adım: Elasticsearch Container Çalıştırın

Elasticsearch container’ını çalıştırın:

docker run -d --name elasticsearch \
-p 9200:9200 -p 9300:9300 \
-e "discovery.type=single-node" \
elasticsearch:8.10.0

3. Adım: Kontrol

Kurulumun başarıyla tamamlandığından emin olun:

curl -X GET "localhost:9200"

Sonuç

Elasticsearch’i ister yerel ister Docker ile kolayca kurabilirsiniz. Her iki yöntem de güçlü log yönetimi ve analiz ihtiyaçlarınız için uygundur.
