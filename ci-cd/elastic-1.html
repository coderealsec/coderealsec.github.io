Rocky Linux'ta Elasticsearch Kurulumu

Rocky Linux üzerinde Elasticsearch kurulumuna başlamadan önce sisteminizin güncel olduğundan emin olun. Güncellemeleri yaptıysanız, hemen başlayalım!

1. Elasticsearch Deposu Ekle

Elasticsearch’i sisteminize tanıtmak için Elastic’in resmi deposunu ekliyoruz. İşte komutlar:


sudo rpm --import https://artifacts.elastic.co/GPG-KEY-elasticsearch
cat <

Not: Bu ayarlar Elasticsearch'in 8.x serisi içindir. Daha yeni sürümleri tercih ediyorsanız Elastic'in GitHub sürüm sayfasını kontrol edin.

2. Elasticsearch'i Kur

Depoyu eklediğimize göre şimdi Elasticsearch’i kurabiliriz:


sudo dnf install elasticsearch

Kurulum hızlı bir şekilde tamamlanacaktır.

3. Servisi Başlat ve Aktif Hale Getir

Elasticsearch artık kurulu! Şimdi bunu çalıştırmamız gerekiyor:


sudo systemctl enable elasticsearch
sudo systemctl start elasticsearch

Servisin başladığından emin olmak için şu komutla durumunu kontrol edebilirsiniz:


sudo systemctl status elasticsearch

4. Bağlantıyı Kontrol Et

Son olarak, Elasticsearch’ün düzgün çalıştığından emin olmak için şu komutu çalıştırın:


curl -X GET "localhost:9200/?pretty"

Eğer her şey yolundaysa, JSON formatında bir çıktı göreceksiniz. Tebrikler, Elasticsearch çalışıyor!

Docker ile Elasticsearch Kurulumu

Docker mı kullanıyorsunuz? Harika! İşte Docker ile Elasticsearch kurulumunun hızlı bir yolu:

1. Docker Compose Dosyasını Hazırla

Elasticsearch için bir docker-compose.yml dosyası oluşturun ve aşağıdaki içeriği ekleyin:


version: '3.7'
services:
  elasticsearch:
    image: docker.elastic.co/elasticsearch/elasticsearch:8.x
    container_name: elasticsearch
    environment:
      - discovery.type=single-node
    volumes:
      - es_data:/usr/share/elasticsearch/data
    ports:
      - "9200:9200"
volumes:
  es_data:
    driver: local

2. Elasticsearch’i Başlat

Şimdi Docker Compose ile Elasticsearch’i ayağa kaldırabiliriz:


docker-compose up -d

Birkaç saniye bekleyin ve her şeyin düzgün çalıştığından emin olmak için şu komutu deneyin:


curl -X GET "localhost:9200/?pretty"
