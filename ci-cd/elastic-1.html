Elasticsearch Kurulumu

1. Elasticsearch Native Kurulumu RHEL Dağıtımları

RHEL Dağıtımları için Elasticsearch'in native kurulumunu gerçekleştirmek için aşağıdaki adımları izleyebilirsiniz:

Adım 1: Java Yükleme

Elasticsearch, Java’ya bağımlıdır. Bu nedenle ilk adım olarak Java'yı kurmanız gerekmektedir. Aşağıdaki komutla Java'yı yükleyebilirsiniz:

sudo dnf install java-11-openjdk

Adım 2: Elasticsearch Deposu Ekleme

Elasticsearch için resmi Elastic deposunu eklemek için aşağıdaki komutları kullanabilirsiniz:


sudo rpm --import https://artifacts.elastic.co/GPG-KEY-elasticsearch
sudo tee /etc/yum.repos.d/elasticsearch.repo <

Adım 3: Elasticsearch’i Kurun

Aşağıdaki komutla Elasticsearch’i yükleyebilir ve çalıştırabilirsiniz:

sudo dnf install elasticsearch
sudo systemctl enable --now elasticsearch

Adım 4: Yapılandırma

Elasticsearch’in çalışma ayarlarını düzenlemek için /etc/elasticsearch/elasticsearch.yml dosyasını açın ve aşağıdaki gibi yapılandırın:


network.host: 0.0.0.0
discovery.type: single-node

2. Elasticsearch Docker Kurulumu

Docker kullanarak Elasticsearch’i izole bir ortamda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

Adım 1: Elasticsearch Docker Image’ını Çekin

Docker Hub’dan Elasticsearch image’ını çekmek için şu komutu kullanın:

docker pull elasticsearch:8.10.0

Adım 2: Elasticsearch Container’ını Başlatın

Elasticsearch container’ını başlatmak için şu komutu kullanabilirsiniz:


docker run -d --name elasticsearch \
-p 9200:9200 \
-e "discovery.type=single-node" \
elasticsearch:8.10.0

Adım 3: Kurulumu Kontrol Edin

Elasticsearch kurulumunun başarılı olduğunu kontrol etmek için şu komutu kullanabilirsiniz:

curl -X GET "localhost:9200"

Sonuç

Bu adımlarla Elasticsearch'i başarıyla kurabilirsiniz. Native kurulum ve Docker kurulum seçenekleri ile sisteminizi esnek bir şekilde yapılandırabilirsiniz.
