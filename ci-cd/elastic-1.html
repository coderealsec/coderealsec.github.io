Elasticsearch Kurulum Rehberi

1. Elasticsearch Native Kurulumu RHEL Dağıtımları

RHEL Dağıtımları için Elasticsearch'in native kurulumunu gerçekleştirmek için aşağıdaki adımları izleyebilirsiniz:

Adım 1: Java Yükleme

Elasticsearch, Java’ya bağımlı olarak çalışır. Bu nedenle ilk adım olarak JAVA JDK kurulması gerekmektedir. Aşağıdaki adım ile paketi kurmamız gerekmektedir.

sudo dnf install java-11-openjdk

Adım 2: Elasticsearch Deposu Ekleme

Elasticsearch için resmi Elastic deposunu sisteminize eklememiz gerekmektedir. Default Epel veya appstream repositoriyde gelmediğinden dolayı elasticsearch'i aşağıdaki adım ile linux repository ekleyebiliriz.

sudo rpm --import https://artifacts.elastic.co/GPG-KEY-elasticsearch
sudo tee /etc/yum.repos.d/elasticsearch.repo <

Adım 3: Elasticsearch’i Kurun

Aşağıdaki komutları kullanarak Elasticsearch’i yükleyin ve başlatın:

sudo dnf install elasticsearch
sudo systemctl enable --now elasticsearch

Adım 4: Yapılandırma

Elasticsearch’in çalışma davranışını düzenlemek için /etc/elasticsearch/elasticsearch.yml dosyasını açarak aşağıdaki gibi yapılandırma yapabilirsiniz:

network.host: 0.0.0.0
discovery.type: single-node

2. Elasticsearch Docker Kurulumu

Docker kullanarak Elasticsearch’i izole bir ortamda çalıştırmak oldukça kolaydır. İşte adımlar:

Adım 1: Elasticsearch Docker Image’ını Çekin

Docker Hub’dan Elasticsearch image’ını çekmek için şu komutu kullanın:

docker pull elasticsearch:8.10.0

Adım 2: Elasticsearch Container’ını Başlatın

Elasticsearch container’ını aşağıdaki komutla başlatabilirsiniz: Elasticsearch aşağıdaki image pull edip container başlatacaktır. Güncel sürümü kurabilmek için docker.hub.com adresinden Elasticsearch sürümünü kontrol edebilirsiniz.

docker run -d --name elasticsearch \
-p 9200:9200 \
-e "discovery.type=single-node" \
elasticsearch:8.10.0

Adım 3: Kurulumu Kontrol Edin

Kurulumun başarıyla tamamlanıp tamamlanmadığını şu komutla kontrol edin:

curl -X GET "localhost:9200"

Sonuç

Yukarıdaki yöntemlerden biriyle Elasticsearch kurulumunuzu tamamlayabilirsiniz. Native kurulum performans avantajı sunarken, Docker kurulum taşınabilirlik ve kolaylık sağlar. İhtiyacınıza göre tercihinizi yapabilirsiniz.
